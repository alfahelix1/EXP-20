Aim
To perform exploratory data analysis on a global COVID-19 dataset in order to understand the spread, recovery, and active case trends across countries — with a focused analysis on India's state-wise distribution — and to visualize the findings using interactive world maps.

Theory
COVID-19 data analysis involves processing time-series records of confirmed, recovered, and death counts across regions. Key concepts applied in this project include:

Data Preprocessing — Removing irrelevant columns, converting data types, and handling missing values to ensure clean, usable data.
Feature Engineering — Deriving the Active case count as Confirmed - Recovered - Deaths to reflect the real-time disease burden.
Aggregation & Grouping — Using groupby operations to summarize case counts at the country and state level.
Snapshot Analysis — Filtering to the latest observation date to get the most current picture of the pandemic.
Geospatial Visualization — Using choropleth maps to represent case distribution visually on a world map, making patterns easier to interpret.


Conclusion
The analysis successfully processed and explored global COVID-19 data. Country-level aggregation revealed significant disparities in confirmed and recovered cases, with Mainland China, the US, and India among the most affected nations. The date-wise trend showed a consistent rise in cumulative cases over time. India's state-level drill-down identified the states with the highest case loads and handled missing province data effectively. Interactive choropleth maps provided a clear visual understanding of both the global spread and recovery patterns, making the analysis accessible and insightful.
